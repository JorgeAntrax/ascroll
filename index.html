Animate CSS & Vanilla JS

Ver en Github Descargar librería
Esta pequeña librería tiene como finalidad ayudarte a implementar animaciones con la librería animate css sin la necesidad de utilizar jquery para aplicar animaciones tomando como referencía el scroll de tu página web.

Como empezar

Por el momento este pequeño script no esta subido a NPM, asi que por ahora descargate el fichero ascroll.min.js y realiza el enlace correspondiente al script.

<script src="js/ascroll.min.js">

Es todo ahora dispones de varias opciones de configuración:

  1. clases
  2. @animate
  3. @scroll

Las clases se añaden de manera normal, excepto la clase que aplica la animación de la librería , revisa la documentación de la librería animate en la cúal dispones de las siguientes opciones:

  1. animated Clase que indica que ese elemento será animado.
  2. infinite Clase que indica que la animacion una vez asignada será infinita.
  3. delay Clases para retrasar la animación

El atributo @animate recibe como valor el nombre de la animación que deseas aplicar a ese elemento

El atributo @scroll recibe como valor un número por ejemplo 200, y significa que cuando el valor de la propiedad scrollY del objeto window sea igual 200 se añadirá la animación que definas en el atributo @animate

Veamos un ejemplo.

Efecto wobble con 400px de scroll

<div class="animated" @animate="wobble" @scroll="400">
    <p>Lorem ipsum dolor sit amet.</p>
</div>

Veamos más ejemplos

Efecto bounceInDown con 100px de scroll

<div class="animated" @animate="bounceInDown" @scroll="100">
    <p>Lorem ipsum dolor sit amet.</p>
</div>

Efecto hinge con 300px de scroll

<div class="animated" @animate="hinge" @scroll="300">
    <p>Lorem ipsum dolor sit amet.</p>
</div>

Efecto slideInUp con 200px de scroll

<div class="animated" @animate="slideInUp" @scroll="200">

</div>
